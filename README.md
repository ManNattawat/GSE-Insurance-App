# GSE Insurance App

แอพพลิเคชันสำหรับเก็บข้อมูลลูกค้าที่ต้องการใช้บริการประกันภัยรถยนต์

## ฟีเจอร์

- กรอกข้อมูลลูกค้าสำหรับการสมัครประกันภัยรถยนต์
- Multi-step form (6 ขั้นตอน)
- อัปโหลดเอกสารประกอบ
- ดูรายการลูกค้า
- ค้นหาและกรองข้อมูล

## การติดตั้ง

```bash
npm install
npm start
```

## เทคโนโลยี

- React Native + Expo
- TypeScript
- Supabase (Database)
- React Navigation
